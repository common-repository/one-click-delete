Automatically deletes a plugin once it gets deactivated.

== Description ==

This plugin automatically deletes other plugins when they get deactivated. This was built out of the need of eliminating clicks. 

== Installation ==

1. Upload `delete-plugins-one-click` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That is it
